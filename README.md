# Mythology Forge v0.3

Consumes `episode_bundle` and produces a richer `media_bundle`.

## New in v0.3
- Reads `scene_prompts.json` if present
- Generates scene frame placeholders from prompt count
- Generates teaser clip placeholders from scene count
- Carries prompt/narration summaries into `media_manifest.json`

## Run
```bash
python forge.py
```
