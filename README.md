# Mythology Forge v0.4

Consumes `episode_bundle` and produces a scene-aware `media_bundle`.

## New in v0.4
- Builds scene-level assets directly from `scene_prompts.json`
- Adds `scene_asset_map.json`
- Maps teaser clips to source scenes
- Emits richer scene metadata in `media_manifest.json`

## Run
```bash
python forge.py
```
